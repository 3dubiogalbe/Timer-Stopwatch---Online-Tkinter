# ⏱️ Cronômetro em Python com Interface Gráfica (Tkinter)

Um aplicativo de cronômetro funcional com interface gráfica simples e intuitiva, desenvolvido em Python utilizando a biblioteca nativa Tkinter. Este projeto faz parte da minha jornada de estudos e consolidação de lógica de programação voltada para automação e DevOps.

## 🚀 Funcionalidades
* **Começar cronômetro:** Inicia a contagem do tempo de forma linear.
* **Pausar cronômetro:** Interrompe a contagem mantendo o registro do tempo atual.
* **Resetar cronômetro:** Zera o contador e interrompe o fluxo para reiniciar do zero.
* **Interface Responsiva:** Botões limpos e organizados com visualização em tempo real.

## 🛠️ Tecnologias Utilizadas
* **Python 3.x**
* **Tkinter** (Biblioteca nativa para interfaces gráficas)
